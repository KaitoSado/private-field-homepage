# サブスク経済

このフォルダは、特別記事や会員限定機能に対する継続課金の設計メモを置く場所です。

## 想定する役割

- 特別記事の閲覧権
- 会員限定機能
- 継続的な収益源
- 公共財や運営資金の下支え

## 想定する対象

- 特別記事
- 限定コミュニティ
- 会員限定 Apps
- 高度な検索・整理・通知機能

## 今の段階で考えるべきこと

- 月額プラン
- 年額プラン
- 無料体験
- ポイントとの併用
- 慶應認証との関係
- 支払い方法
- 解約と権限剥奪

## 実装上の有力候補

- Stripe Checkout
- Customer Portal
- Webhook による権限同期

## 注意

- サブスクは `信用` ではなく `権利` に近い
- ポイント経済と混ぜすぎない
- 称号や評価は課金で買えないようにする
