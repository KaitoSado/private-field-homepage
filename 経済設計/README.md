# 経済設計

このフォルダは、このサービスにおける経済圏の設計をまとめる場所です。

## 構成

- [ポイント経済](</Users/sadokaito/Downloads/homepage/経済設計/ポイント経済/README.md>)
  - 利用ポイント、貢献度、信頼赤字、称号、共同金庫、取引台帳
- [仮想通貨経済](</Users/sadokaito/Downloads/homepage/経済設計/仮想通貨経済/README.md>)
  - 将来的なガバナンストークン、分配、ロックアップ、投票権
- [サブスク経済](</Users/sadokaito/Downloads/homepage/経済設計/サブスク経済/README.md>)
  - 特別記事、会員権、支払い導線、継続課金、限定機能

## 全体方針

- 初期は `ポイント経済` を中心に回す
- `仮想通貨経済` はかなり後段で検討する
- `サブスク経済` は現実的な収益化の中核として先に設計してよい
- どの経済圏でも、`信用` と `決済` を同一視しない

## このサービスで守る原則

- ギバーが報われる
- テイカーだけが得し続けない
- 金の流れと信頼の流れを分ける
- 公共財への貢献がちゃんと報酬につながる
- 慶應認証など信頼軸を制度に取り込む
