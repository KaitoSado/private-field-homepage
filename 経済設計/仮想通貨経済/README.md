# 仮想通貨経済

このフォルダは、将来的にこのサービスで仮想通貨またはガバナンストークンを扱う場合の設計メモを置く場所です。

## 想定する役割

- ガバナンス参加権
- 収益分配の設計
- 長期貢献者への報酬
- ポイント経済より上位の制度参加権

## 想定する前提条件

- ポイント経済が安定している
- 不正検知と信用評価が機能している
- サブスクや共同金庫の運用実績がある
- 初期ユーザー群の経済行動データが蓄積している

## 今の段階で考えるべきこと

- ガバナンストークンの目的
- 配布条件
- ロックアップ
- 投票権の重み
- 社会的資本との連動

## 注意

- 現段階ではまだ実装優先度は低い
- 先にポイント経済とサブスク経済を固める
