## このサービスでの基本方針

- `利用ポイント` と `信用・称号` を分ける
- 情報共有、助け合い、売買、依頼を同じ経済圏で扱う
- ギバーが報われ、テイカーだけが得し続けない構造をつくる
- 初期は閉じたポイント経済で回し、複雑な金融化は後段に送る

## まず制度として必要なもの

- 発行ルール
- 流通ルール
- 回収と焼却
- 共同金庫
- 貢献度の減衰
- 称号
- 不正検知
- 慶應認証との連動
